# NumberSorter
Sorts 4 values from highest to lowest.

**Directions:**  
Fill out the 4 `{set}`s at the begining of the snippet with the values you want sorted.  
The snippet will sort the values and place them in keys.  
`{get;1}` - Highest  
`{get;2}` - Second Highest  
`{get;3}` - Second Lowest  
`{get;4}` - Lowest
