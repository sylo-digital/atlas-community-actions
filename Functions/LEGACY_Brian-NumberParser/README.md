# Number Parser
This function parses your input into a valid number.

**Examples:**
* **Input:** `/parse parameter:100k`\
  **Output:** `100000`

* **Input:** `/parse parameter:420m`\
  **Output:** `420000000`

* **Input:** `/parse parameter:69b`\
  **Output:** `69000000000`
  
* **Input:** `/parse parameter:56.4k`\
  **Output:** `56400`
